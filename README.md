## Build instructions

```
$ git clone https://github.com/patrickhadlaw/patrickhadlaw.com
$ cd patrickhadlaw.com/frontend
$ npm install -g @angular/core
$ npm install -g @angular/cli
$ npm install
```

## Run instructions: 

```
$ cd patrickhadlaw.com/frontend
$ ng serve
```
